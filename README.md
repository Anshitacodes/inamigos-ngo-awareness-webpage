# InAmigos Foundation - NGO Awareness Webpage

## Project Overview

This project is a responsive NGO Awareness Webpage developed using HTML and CSS for the InAmigos Foundation.

The purpose of this webpage is to spread awareness about the foundation's mission, ongoing projects, social impact, and volunteer opportunities.

---

## Features

- Introduction about InAmigos Foundation
- Information about ongoing projects
- Social impact and purpose of initiatives
- Image gallery showcasing activities
- Call-to-action section
- Responsive design
- Professional blue-themed user interface

---

## Key Initiatives Covered

### Project SEVA
Providing food, clothing, and essential support to underprivileged communities.

### Project BACHPANSHALA
Supporting education, digital literacy, and learning opportunities for children.

### Project JEEV
Promoting animal welfare through rescue, feeding, and protection initiatives.

### Project UDAAN
Empowering women through skill development and financial independence.

### Project PRAKRITI
Encouraging environmental sustainability through plantation and conservation drives.

### Project VIKAS
Providing skill development and internship opportunities for youth.
